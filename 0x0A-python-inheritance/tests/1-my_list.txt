=======================
How to Use 1-my_list.py
=======================

This module defines a class ``MyList`` that inherits from ``list``.

Instantiation
=============

``MyList`` can be instantiated with no arguments, which will return
an empty list:

::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> type(my_list)
    <class '1-my_list.MyList'>

::

    >>> print(my_list)
    []

::

# Test Case 2: List with integers

	>>> MyList([1, 4, 7, -3, 9])
	[-3, 1, 4, 7, 9]

::

# Test Case 3: List with strings

	>>> MyList(['orange', 'grape', 'apple', 'banana'])
	['apple', 'banana', 'grape', 'orange']

::

# Test Case 4: List with mixed types (integers and strings)

	>>> MyList(['apple', 'banana', 7, -3, 'grape', 1, 9])
	[-3, 1, 'apple', 'banana', 'grape', 7, 9]

::

# Test Case 5: List with duplicates

	>>> MyList([5, 4, 3, 2, 1, 4, 2, 5])
	[1, 2, 2, 3, 4, 4, 5, 5]

::

# Test Case 6: List with a single element

	>>> MyList([42])
	[42]
