The ``MyList`` class
====================

Creating a MyList instance
---------------------------

Let's create a MyList instance with an unsorted list:

    >>> my_unsorted_list = MyList([3, 1, 4, 1, 5, 9, 2])

Now, print the unsorted list using the print_sorted method:

    >>> my_unsorted_list.print_sorted()
    [1, 1, 2, 3, 4, 5, 9]

Sorting an already sorted list
-------------------------------

Now, let's create a MyList instance with a pre-sorted list:

    >>> my_sorted_list = MyList([1, 2, 3, 4, 5, 6])

Print the already sorted list using the print_sorted method:

    >>> my_sorted_list.print_sorted()
    [1, 2, 3, 4, 5, 6]

Sorting an empty list
---------------------

Let's create an empty MyList instance:

    >>> my_empty_list = MyList()

Print the empty list using the print_sorted method:

    >>> my_empty_list.print_sorted()
    []

Sorting a list with duplicate elements
---------------------------------------

Now, create a MyList instance with a list containing duplicate elements:

    >>> my_list_with_duplicates = MyList([2, 4, 1, 3, 2, 4, 3, 5])

Print the list with duplicates using the print_sorted method:

    >>> my_list_with_duplicates.print_sorted()
    [1, 2, 2, 3, 3, 4, 4, 5]

Sorting a list with mixed types
-------------------------------

Create a MyList instance with a list containing mixed data types (integers and strings):

    >>> my_mixed_type_list = MyList([3, 'apple', 1, 'orange', 4, 'banana'])

Print the mixed type list using the print_sorted method:

    >>> my_mixed_type_list.print_sorted()
    [1, 3, 4, 'apple', 'banana', 'orange']
