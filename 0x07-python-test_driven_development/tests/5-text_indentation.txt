# Valid Test Cases

# Test Case 1
# Simple sentence without new lines
This is a simple sentence.
Expected Output:
This is a simple sentence.

# Test Case 2
# Sentence with multiple spaces and new lines
   This   is a sentence.
With multiple spaces and new lines.
Expected Output:
   This   is a sentence.$
$
With multiple spaces and new lines.

# Test Case 3
# Sentence with multiple characters triggering new lines
This is a sentence? A new sentence: yes!
Expected Output:
This is a sentence?$
A new sentence:$
yes!

# Test Case 4
# Empty string
""
Expected Output:
(prints nothing)
